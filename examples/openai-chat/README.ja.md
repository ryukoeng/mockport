# OpenAI Chat Example 日本語版

[English](README.md)

この example は、OpenAI-compatible local endpoint に chat request を送り、実 API key なしで AI application の integration path を確認するためのものです。

## 確認すること

- `OPENAI_BASE_URL` を Mockport の local endpoint に向けること。
- fake API key を使うこと。
- deterministic fake response と error scenario を確認すること。

# Code of Conduct 日本語版

[English](CODE_OF_CONDUCT.md)

Mockport への参加者は、敬意を持った協力的なコミュニケーションを前提にします。議論では、個人攻撃ではなく具体的な技術的事実、再現手順、改善提案を優先してください。

## 基本方針

- issue、PR、review では相手を尊重し、根拠のある指摘を行います。
- 差別、嫌がらせ、威圧的な言動、公開の場での個人情報共有は許容しません。
- 問題がある場合は maintainer に連絡し、公開 thread での応酬を広げないでください。

正式な適用範囲と enforcement は英語版を正とします。

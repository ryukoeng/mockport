# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and direct.
- Keep discussion focused on the technical issue.
- Assume public issues and pull requests may be read by new users evaluating the project.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information or real secrets.
- Repeatedly derailing technical discussion.

## Enforcement

Maintainers may edit, hide, or remove comments and may block users who violate this code of conduct.

# Mockport Development Documentation Pack 日本語版

[English](README.md)

この directory は、Mockport 初期設計時の product concept、MVP scope、Go 実装方針、architecture、adapter model、CLI、Docker、security、roadmap をまとめた開発ドキュメント pack です。

## 位置づけ

- 現行の public docs は `docs/` と `docs/site/` を優先します。
- この pack は設計背景と初期判断を追うための補助資料です。
- 古い前提が残る場合は、現行 README、support matrix、compatibility report を正とします。

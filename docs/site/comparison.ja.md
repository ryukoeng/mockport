# Comparison 日本語版

[English](comparison.md)

Mockport は provider sandbox、mock server、contract test tool の完全な置き換えではありません。AI-native development と CI で、secret-free に selected workflow を検証するための local emulator です。

## 比較軸

- 実 provider sandbox より安全に fake credential で動かせます。
- 汎用 mock より adapter-specific な workflow と report を持ちます。
- full fidelity より、supported scope と known gap の明示を優先します。

# MVP Scope 日本語版

[English](02_mvp_scope.md)

初期 MVP は Stripe-like adapter を中心に、Docker で起動できる local emulator と CLI bootstrap を提供する範囲に絞ります。

## 範囲

- fake credential と local endpoint。
- checkout success/failure などの基本 scenario。
- AI-safe warning と report。
- provider full fidelity や production validation は対象外です。

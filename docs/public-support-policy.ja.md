# Public Support Policy 日本語版

[English](public-support-policy.md)

Mockport の public support は、supported adapter workflow、known gap、compatibility evidence を明示することを重視します。

## 方針

- 対応済みと未対応を support matrix で分けます。
- provider の undocumented behavior は再現保証しません。
- issue では adapter、workflow、expected/actual behavior、report output を含めてください。

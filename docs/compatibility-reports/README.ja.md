# Compatibility Reports 日本語版

[English](README.md)

compatibility reports は、adapter ごとの support claim、SDK/client evidence、provider API version、known gap を確認するための生成物です。

## 使い方

- `latest.md` で人間向けの最新 summary を確認します。
- `latest.json` で machine-readable な evidence を確認します。
- adapter maturity を上げる場合は、report と source docs を一緒に更新します。

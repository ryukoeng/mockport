# v0.1.0-alpha Release Notes 日本語版

[English](v0.1.0-alpha.md)

`v0.1.0-alpha` は Mockport の最初の public preview release です。Docker-first distribution と selected adapter workflow の local/CI 検証を主目的にしています。

## 確認ポイント

- Docker image と GitHub release archive が primary distribution です。
- adapter support は selected workflow に限定されます。
- `latest` tag は default branch に追従し、preview release contract ではありません。

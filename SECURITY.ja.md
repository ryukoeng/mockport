# Security Policy 日本語版

[English](SECURITY.md)

Mockport は実 provider の secret を不要にすることで、AI coding や CI での誤送信リスクを下げることを目的にしています。脆弱性や secret exposure の疑いを見つけた場合は、公開 issue に機密情報を書かず maintainer に報告してください。

## 報告時の注意

- 実 token、customer payload、private endpoint は貼らないでください。
- 再現手順、影響範囲、該当 commit または version を共有してください。
- public examples に real-looking secret が混入している場合は security issue として扱います。

サポート対象 version、報告先、対応方針は英語版を正とします。
